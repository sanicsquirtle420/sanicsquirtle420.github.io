# SanicSquirtle428's GitHub Pages
## Development
Setting up envoirnment
```
npm create vite@latest
npm install
```
Useful commands
```
npm run dev // Starts Vite
npm run predeploy // Compiles everything for GitHub pages
npm run deploy // Sends to GitHub pages
```
